# Technology Stack

## Anaconda Navigator
Used to manage Python environments and launch Jupyter Notebook for machine learning model development.

Official Website:
https://www.anaconda.com/download

## PyCharm
Used as a Python IDE for coding, debugging, and application development.

Official Website:
https://www.jetbrains.com/pycharm/

## NumPy
Used for numerical computations and array operations.

Official Website:
https://numpy.org/

## Pandas
Used for data loading, preprocessing, and analysis.

Official Website:
https://pandas.pydata.org/

## Scikit-learn
Used to train and evaluate the Random Forest machine learning model.

Official Website:
https://scikit-learn.org/

## Matplotlib
Used for plotting graphs and data visualization.

Official Website:
https://matplotlib.org/

## Seaborn
Used for statistical data visualization.

Official Website:
https://seaborn.pydata.org/

## Flask
Used to develop and deploy the Loan Approval Prediction web application.

Official Website:
https://flask.palletsprojects.com/
